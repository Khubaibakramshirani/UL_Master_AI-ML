Task 3 - Naïve Bayes Classifier

Weight: 20/100

The Naïve Bayes Classifier is a simple probabilistic classifier that is based on the application of the Bayes theorem. In this task, we will not code the Naïve Bayes Classifier by hand, but instead will employ the powerful Python library, scikitlearn. We will also use the pandas library to read the data from the CSV. 

We will train an algorithm to predict a person’s sex based on the features of height (in inches) and weight (in pounds). We will use a large set of samples to train our classifier. 

  1. Plot the data from the CSV file to see if there is a reasonable chance of separating the sexes using our two selected features. Does it seem that our classifier has some chance of getting a good result? 
    1. Examining data is very important. If there was no chance of classification (for example, if the data were randomly mixed and no evident pattern), then we would need a new classification approach or a new set of features
    2. You can use matplotlib for the plotting, or pandas own plotting functions (ref data.plot)
  2. Use the train_test_split function in scikitlearn to split the data into training and test data. Use a split of 33% for test data and the remainder for training.  
    1. The importance of splitting training and test data will be explained later in the module 
  3. Fit a model using GaussianNB from scikitlearn. Train using the train data. Test using the predict function on the test data. A Gaussian model is required for continuous variables (such as height and weight) 
  4. What accuracy do you get? (Hint: accuracy_score from sklearn.metrics) 
  5. Create lots (e.g. 5000) of samples of persons with random weights and heights between reasonable ranges. Apply the predict function to predict if these random feature pairs are male or female. Plot on a scatter plot 

The last part of the exercise will give you a good feeling about the decision boundary of the classifier.
